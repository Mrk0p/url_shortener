# URL Shortener

A simple, fast, and efficient URL Shortener built using [Vite](https://vitejs.dev/). This application allows users to shorten long URLs, track usage, and manage their short links easily.

## Features

- Generate short URLs from long URLs.
- Custom aliases for URLs.
- Track usage and clicks.
- Simple and intuitive UI.
